# book-review-api
In this final project, you will develop a server-side web application for online book reviews. The application will allow users to browse books, write reviews, and interact with other users' feedback.
# مشروع مراجعة الكتب (Book Review App)

## وصف المشروع
هذا المشروع هو تطبيق **مراجعة الكتب** باستخدام **Node.js وExpress وJWT**.  
يسمح التطبيق للمستخدمين بعرض الكتب المتاحة، البحث عنها حسب ISBN أو المؤلف أو العنوان، وإضافة أو تعديل أو حذف المراجعات إذا كانوا مسجلين.  
كما يستخدم المشروع **JWT وSession** للتحقق من هوية المستخدم وحماية نقاط النهاية.

---

## المتطلبات
- Node.js >= 14
- npm
- Postman لاختبار Endpoints
